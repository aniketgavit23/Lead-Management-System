# Lead Management System

This project was developed by  Aniket Gavit and Harshvardhan for academic submission.

Project Name: Lead Management System

A professional Java-based Lead Management System for tracking sales leads with automatic scoring and analytics.

## 🚀 Features

- ✅ **Complete CRUD Operations** - Create, Read, Update, Delete leads
- 🔍 **Smart Search** - Search by name, email, or company
- 📊 **Auto Lead Scoring** - Automatic scoring based on company type and lead quality
- 📈 **Analytics Dashboard** - View statistics and lead distribution
- 💾 **File-based Storage** - No database setup required
- 🎯 **Pipeline Management** - Track leads through NEW → CONTACTED → QUALIFIED → WON/LOST

## 🛠️ Tech Stack

- Java 17+
- File I/O for persistence
- Collections Framework
- MVC Architecture Pattern

## 📋 Prerequisites

- Java 17 or higher installed
